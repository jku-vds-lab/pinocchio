Statistics

Introduction to the general problem

Modern Science is mostly based on Data. Recently especially during the Covid19 Pandemic we saw how important it is to base ones decision on data. The problem of course is, when you can show data which seems related it does not at all mean that there is indeed a relationship.

Spurious Relationsships

While it is important to see that data-based decision making is very important no matter which domain we are talking about. It could be medicine, research, business, economics or policy!
But one problem remains. Some basic measure to see if something has some relationship we are using correlation. Correlation shows just if variable x increases also variable y increases so one could think they are related.

Statistical - Significance

To get a basic introduction what statistical tests are and how they work look at the corresponding page!

Statistical Tests are one kind of technology to check if some observation is found by random or if this oberservation is so clear that the relationship must be there!

Basically one has significance niveau 95 if one can say that when you repeat the experiment 100 times you will still have the same result in 95 % of the cases.

Multiple Statistical Tests as a Trap

Assume now that you have a huge dataset. Think on big companies like Google or the government. You see lots of data!
So now lets say that if you have some intuition or have some domain knowledge about a topic and you know that something might be related.
Ok, you check, YEAH there is a significant relation.
But now you are tired and correlate just any things you can. And now lets say we have a certain probability (very small) that things are significantly correlated.
P = 0.005
So and now you correlate: P + P + P + ...
You see the probability increases when doing many tests and its very likely at some point to find results such that they are significantly correlated.
And here there comes the point: Correlation and Causation: Correlation: if some variable increases the other increases/decreases as well. Causation: when some variable changes it directly affects the world and some other thing is moving. So when observing some correlation, some regression there are three options.

  • Causality - X causes Y
  • Backward Causality - Y causes X
  • Suprious Relationship - X and Y move when changing the other but actually there is no relation.
